Contest Information

Rules

  • General contest rules for all ACM-ICPC contests are detailed as by ICPC Regional Rules for 2016 and Asia Director's Blog.
  • Specific contest rules for the 2016 ACM-ICPC Chung-Li Regional Contest:
    • Each team may bring up to 25 printed pages of A4 or letter sized printed document, with page numbers clearly labeled.
    • Each team may bring necessary stationery (e.g., pen, pencil, eraser, ruler, etc.), a printed English dictionary, and a team mascot.
    • Any electronic devices such as smart phones, personal computers, USB flash disks, memory cards, portable disks, and programmable calculators should not be taken into the contest. Exception: Keyboards and mouses can be taken into the contest if being checked by the contest staffs.
    • The contest staffs can check anything brought into the contest at any time before and during the contest.

Environment

  • The 2016 ACM-ICPC Chung-Li Regional Contest uses CSUS PC^2 9.2.5 as its judgement system.
  • Each team will be allocated with a personal computer (with unified hardware and software configurations) running Linux.
  • Hardware configuration: CPU Intel Core i5-4460 3.2 GHz (Quad core), 8GB DRAM.
  • Software environment: Ubuntu Desktop 15.10 (64-bit) with Unity desktop; text editors: vi / Vim / gVim / Emacs / gedit / Geany / Sublime Text; contest control system: CSUS PC^2 9.2.5.
  • Integrated development environments (IDEs): Eclipse 4.5.2 and Code::Blocks 13.12.
  • The available programming languages include C, C++, and Java. Compilers for the languages include: Java 1.8.0_91, GCC/G++ 5.2.1.
  • C compiler commands: gcc -g -O2 -std=gnu99 -static -lm -o <exe> <source>; Runner: <exe>
  • C++ compiler: g++ -g -O2 -std=gnu++11 -static -lm -o <exe> <source>; Runner: <exe>
